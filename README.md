## 📌 Overview  
Lyriks is a modern music player web application built using React, designed to allow users to explore, play, and manage music with real-time API data and smooth UI interactions.  
It focuses on performance, state management, and seamless user experience.

<p align="center">
  <img src="https://github.com/aliiashraf777/1rapi-lyriks-music-player/blob/main/src/assets/24pr24preview.png" />
</p>

## 🎯 Problem  
- ❌ Complex and heavy music platforms  
- ❌ Poor UI responsiveness  
- ❌ Difficult state management for playback  
- ❌ Inefficient data handling  

## 💡 Solution  
- ✅ Clean and modern music UI  
- ✅ Efficient global state management  
- ✅ API-powered dynamic music data  
- ✅ Smooth playback experience  

## ⚡ Key Features  
- 🎵 Music browsing system  
- ▶️ Audio player controls  
- 🔍 Search and discovery  
- 📊 Dynamic music data (API)  
- 📱 Fully responsive design
